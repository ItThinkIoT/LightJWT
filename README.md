# LightJWT

This is a helper library to create Json Web Token [JWT](https://jwt.io/) 

### Suported Algorithms
  
- RS256

### Features

- Create JWT
- Base64(UrlSafe) Encode&Decode

### Dependencies

- Arduino
- mbedtls
